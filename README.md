# Link Vink

## Introduction

Welcome to **Link Vink** - a platform where users can unify all their social media and professional accounts into one easily shareable link. Instead of sharing multiple links, users can simply share a single "link-vink.com/username" which displays all their profiles in one place.

## Features

- **Unified Profile**: Combine all your social media and professional profiles into one link.
- **Easy Sharing**: Share just one link for all your online presence.
- **Customizable Profiles**: Users can personalize their profiles with themes and custom links.
- **Responsive Design**: Optimized for both mobile and desktop viewing.

## Tech Stack

- **Frontend**: React.js
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Version Control**: Git

## Getting Started

### Prerequisites

- Node.js
- npm or yarn
- MongoDB

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/link-vink.git
   cd link-vink
   ```

2. **Install server dependencies**:
   ```bash
   cd server
   npm install
   ```

3. **Install client dependencies**:
   ```bash
   cd ../client
   npm install
   ```

### Running the Application

1. **Start the server**:
   ```bash
   cd server
   npm start
   ```

2. **Start the client**:
   ```bash
   cd ../client
   npm start
   ```

3. **Visit the application**:
   Open your browser and go to `http://localhost:3000`

### Configuration

- **Environment Variables**: Create a `.env` file in the `server` directory and add the following:
  ```
  PORT=5000
  MONGO_URI=your_mongodb_connection_string
  ```
